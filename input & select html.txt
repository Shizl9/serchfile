بحث عن عنصري Input و Select في HTML
المقدمة

تُعد لغة HTML من أهم لغات تصميم صفحات الويب، حيث تُستخدم في إنشاء هيكل الصفحة وتنظيم عناصرها. ومن أهم عناصر HTML عناصر النماذج (Forms) التي تسمح للمستخدم بإدخال البيانات والتفاعل مع الموقع. ومن أبرز هذه العناصر: input و select، حيث يُستخدم كل منهما لأغراض مختلفة في إدخال واختيار البيانات.

أولًا: تعريف عنصر Input

عنصر input في HTML يُستخدم لإدخال البيانات من المستخدم، مثل كتابة النصوص أو الأرقام أو البحث داخل الموقع. ويتميز عنصر input بالمرونة، حيث يمكن تغيير نوعه حسب الاستخدام.

استخدام Input كبحث

يمكن استخدام input كحقل بحث يسمح للمستخدم بكتابة كلمة أو نص للبحث عنه داخل الموقع.

مثال على Input كبحث:
<input type="search" placeholder="ابحث هنا...">

أنواع شائعة لعنصر Input
<input type="text">       لإدخال نص
<input type="password">  لإدخال كلمة مرور
<input type="email">     لإدخال بريد إلكتروني
<input type="number">    لإدخال أرقام
<input type="search">    للبحث

ثانيًا: تعريف عنصر Select

عنصر select في HTML يُستخدم لإنشاء قائمة منسدلة تحتوي على مجموعة من الخيارات الجاهزة، ويقوم المستخدم باختيار عنصر واحد أو أكثر منها دون الحاجة إلى الكتابة.

مثال على Select بسيط
<select>
  <option>مصر</option>
  <option>السعودية</option>
  <option>الإمارات</option>
</select>

أنواع واستخدامات عنصر Select
1. Select عادي

لاختيار عنصر واحد فقط.

<select>
  <option>ذكر</option>
  <option>أنثى</option>
</select>

2. Select متعدد الاختيار

لاختيار أكثر من عنصر.

<select multiple>
  <option>HTML</option>
  <option>CSS</option>
  <option>JavaScript</option>
</select>

3. Select مع خيار افتراضي
<select>
  <option disabled selected>اختر الدولة</option>
  <option>مصر</option>
  <option>المغرب</option>
</select>

الفرق بين Input و Select

عنصر input يسمح للمستخدم بكتابة البيانات يدويًا.

عنصر select يوفّر قائمة جاهزة للاختيار.

يُستخدم input في البحث وإدخال النصوص.

يُستخدم select لتقليل الأخطاء وتنظيم الاختيارات.

استخدام Input و Select معًا

في بعض الحالات يمكن الجمع بين input و select لإنشاء حقل بحث مع قائمة اختيارات.

مثال:
<input type="search" list="items" placeholder="ابحث واختر">

<datalist id="items">
  <option value="HTML">
  <option value="CSS">
  <option value="JavaScript">
</datalist>

الخاتمة

في الختام، يُعتبر عنصرا input و select من العناصر الأساسية في HTML، حيث يساعدان في تسهيل إدخال البيانات والتفاعل مع المستخدم. ويعتمد اختيار العنصر المناسب على طبيعة الاستخدام، سواء كان إدخال نص أو البحث أو اختيار قيمة من قائمة جاهزة.
